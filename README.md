# 個人網站作業

這是一個可直接上傳到 GitHub 並部署到 Vercel 的靜態網站。

## 檔案結構
- `index.html`：首頁 / 關於我
- `plan.html`：第二頁 / 我的規劃
- `style.css`：共用樣式

## 本機查看
直接用瀏覽器打開 `index.html` 即可。

## 上傳 GitHub
1. 在 GitHub 建立新 repository
2. 把這三個檔案上傳上去
3. 確認 repository 內有：
   - `index.html`
   - `plan.html`
   - `style.css`

## 部署到 Vercel
1. 登入 Vercel
2. 點選 `Add New...` → `Project`
3. 匯入你的 GitHub repository
4. 這是靜態網站，不需要額外框架設定
5. 點 `Deploy`

## 之後可自行修改的地方
- 文字內容
- 顏色
- 圓角按鈕樣式
- 圖片與裝飾
